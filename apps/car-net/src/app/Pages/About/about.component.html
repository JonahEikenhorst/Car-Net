About the page

The Car-Network webapplication

..is a platform for user to create their own garage with different cars. These cars have different properties that can be viewed when clicked on a car. The users can add and remove any car they'd like to or from their garage. It is also possible to like another person's garage and get recommended garages based on the likes you've given.

To enjoy the full experience of the application, you can create an account and login to your own garage.

Use Cases

Path from user to system to user

UJ.100A: As a user, I want to be able to open my own garage, so that I can see my own cars.
UJ.100B: As a user, I want to be able to add cars to my garage, so I can fill up my garage.
UJ.100C: As a user, I want to be able to remove cars from my garage, so I can sell my cars.

UJ.101A: As a user, I want to be able to see other users garages, so I can see what cars they have.
UJ.101B: As a user, I want to be able to like other users garages, so I can show my appreciation.
UJ.101C: As a user, I want to be able to see recommended garages, so I can see what cars other users have.

UJ.102A: As a user, I want to be able to see the catalog, so I can see what cars are available.
UJ.102B: As a user, I want to be able to buy cars from the catalog, so I can add them to my garage.
UJ.102C: As a user, I want to be able to see the details of a car, so I can see what the car looks like.

UJ.103A: As a user, I want to be able to login, so I can access my own garage.
UJ.103B: As a user, I want to be able to register, so I can create my own garage.
